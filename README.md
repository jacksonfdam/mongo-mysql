### Mongo Mysql Test Performance in Nodejs

#### Installation

```
git clone https://github.com/jacksonfdam/mongo-mysql.git
cd mongo-mysql
npm install
node index
```

Comparison tests made by this repo 

With data size in rows numbers : 

10 rows
```
mysql insert: 1702ms
mysql select: 11ms

mongo insert: 47ms
mongo select: 12ms
```

100 rows
```
mysql insert: 8171ms
mysql select: 10ms

mongo insert: 167ms
mongo select: 60ms
```


1.000 rows
